
# Project Setup Guide


## Getting the Project Up and Running

1. Install dependencies (use legacy peer deps for compatibility):

```bash
npm install --legacy-peer-deps
```

2. Start the development server (for local development only, **not** for production):

```bash
npm run dev
```
